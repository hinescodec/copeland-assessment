To run the application:

1. Clone the repository.
2. Set up your OpenWeatherMap Base URL in a .env file 
(follow the format: REACT_APP_OPENWEATHERMAP_BASE_URL=base-url).
3. Set up your OpenWeatherMap API key in a .env file 
(follow the format: REACT_APP_OPENWEATHERMAP_API_KEY=your-api-key).
4. Run `npm install` to install dependencies.
5. Run `npm run start` to start the application.
6. Open http://localhost:3000 in your browser.

To run tests:

1. Run `npm run test`.
